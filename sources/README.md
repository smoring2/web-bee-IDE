## webBee 为乐趣而爬
 
- `webBee` 基于`jdk8` 是一个持续成长的`垂直爬虫框架`项目 
- `webBee` 遵循`MIT`开源协议

- `webBee` 是一个不错的java进阶项目


- 欢迎大家贡献代码，如果觉得这个项目不错，请为它 `点赞`





### 演示站点
- http://gitrue.com (备案成功，开发中)


### 设计初衷

+ 设计简洁，面向接口
+ 配置简单，明了
+ 一次配置，自动匹配


### 涉及技术栈
- `httpcomponents`
- `jsoup`
- `redis`
-  `代理`
- `多线程`

 
+ 分布式爬取
+ 代理接口
+ 通过配置智能寻找目标 `url`
+ 图片、视频的处理
